LU ROSE GOLD

About WEBSITE OTHER

I designed and developed the Lucinda Rose Gold Tattoo Visualizer, a custom web application for San Francisco-based tattoo artist Lucinda Rose Gold. Built in Three.js, the tool enables clients to preview Lu's designs on a range of 3D models, exploring different placements, sizes, and orientations to better envision their tattoos before committing. The application features intuitive drag-and-drop placement, skin tone selection, size and rotation adjustments, and keyboard and mouse controls for precise model interaction. Users can also share their customized designs directly with the artist, streamlining the consultation process and creating a more confident, collaborative client experience.

Tattoo Visualizer Development

The web application was built using Three.js to create an interactive 3D environment where clients can experiment with tattoo placement in real-time. Key features include customizable skin tones, precise placement controls, and the ability to rotate and scale designs to match the client's vision. This tool transforms the traditional consultation process into an engaging, interactive experience.

Additional design work and collaborations with Lu Rose Gold are coming soon.
